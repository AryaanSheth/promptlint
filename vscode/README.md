# PromptLint VS Code Extension

*(Placeholder for future extension.)*

Planned: real-time diagnostics for prompt files (markdown/plaintext) by running the PromptLint CLI and surfacing findings in the editor.

- **CLI dependency:** This extension will call the [PromptLint CLI](../cli/) (Python); users will need the CLI installed.
- **Scope:** Lint on save or on demand, optional apply-fix from editor.

To contribute or scaffold the extension, add `package.json`, `src/extension.ts`, and follow the [VS Code extension guide](https://code.visualstudio.com/api).
